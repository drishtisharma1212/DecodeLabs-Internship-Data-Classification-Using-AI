# Data Classification Using AI
This project implements a supervised machine learning model to classify Iris flower species based on sepal and petal measurements. The model is trained using Logistic Regression and evaluated on a test dataset to measure its performance.

## Features
- Load and preprocess the Iris dataset
- Split data into training and testing sets
- Train a Logistic Regression classification model
- Evaluate model accuracy
- Predict flower species using user-provided measurements
- Interactive command-line interface

## Technologies Used
- Python 
- Scikit-learn
- Machine Learning
- Logistic Regression

## Dataset
The project uses the Iris Dataset, a widely used dataset for machine learning classification tasks.

## Input Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Output Classes
- Setosa
- Versicolor
- Virginica

## Installation
1. Clone the repository: git clone <repository-url>
2. Open the project folder: cd DecodeLabs-Internship-Data-Classification-Using-AI
3. Install the required dependencies: pip install -r requirements.txt
4. Run the chatbot: python data_classification.py
